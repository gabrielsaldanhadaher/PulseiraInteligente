Projeto para a criação de um site e um sistema para armazenar dados dentro de uma pulseira inteligente.
Essa pulseira armazenará dados de treinos esportivos de pessoas.
A o projeto terá como obejetivo armazenar os dados, trata-los com segurança e informar a evolução do treino de um indivíduo.
O site tem como proposta oferecer os planos para os usuários e mostrar como a pulseira funciona.
